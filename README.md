# 💧 WaterWay

**WaterWay** — это 2D-головоломка, в которой нужно соединять трубы, чтобы провести жидкость из начальной точки в конечную. Проект разработан на Godot Engine 4.4 с фокусом на мобильные устройства и ОС «Аврора».

## ✨ Ключевые особенности
- 📱 **Сенсорное управление:** тап для поворота труб
- 🧩 **Прогрессивная сложность:** 20+ уровней с постепенным введением новых механик
- 🔒 **Уникальные элементы:** заблокированные трубы-вентили, обязательные чекпоинты, система подсказок (2 на уровень)
- 💾 **Локальное сохранение:** прогресс, звёзды и настройки хранятся на устройстве
- ⚡ **Оптимизация:** стабильная работа на мобильных процессорах,ОС «Аврора» и Android

## 🕹️ Как играть
1. Нажмите на трубу, чтобы повернуть её на 90°.
2. Выстройте непрерывный путь от начальной точки до конечной.
3. Убедитесь, что все активные выходы труб соединены (разрывы не допускаются).
4. Используйте кнопку 💡 для подсветки проблемных участков (лимит: 2 подсказки на уровень).
5. Проходите уровни последовательно, открывая новые испытания.

## 📦 Установка и запуск
### ▶️ Готовая сборка
Скачайте последнюю версию в разделе [Releases](https://github.com/top-it-090304/EAgames/releases). Установка стандартная для целевой ОС.

### 🛠 Запуск из исходников
1. Установите [Godot 4.4+](https://godotengine.org/download/)
2. Клонируйте репозиторий:  
   `git clone https://github.com/top-it-090304/EAgames.git`
3. Откройте `project.godot` в редакторе
4. Нажмите `F5` для запуска в режиме отладки
5. Для экспорта: `Проект → Экспорт → Android/Aurora`
